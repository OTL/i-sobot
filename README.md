i-sobot arduino
==================
[Original hacked information](http://robot.watch.impress.co.jp/cda/column/2007/11/08/731.html) is not my work (Japanese).

Servo information
---------------------
* Red -> Power (5V)
* Blue -> GND
* Green -> Rx (Serial 3.3V)
* Orange -> 3.3V

You need level converter if you want to use 5V arduino.
